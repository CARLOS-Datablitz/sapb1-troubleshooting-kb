Solución: Desplegar TS01 como segundo RD Session Host y configurar el broker para balanceo de carga.

04
Script principal — Usuarios conectados hoy
Resumen diario con total de usuarios únicos y detalle de eventos
⭐ Script principal de diagnóstico diario. Ejecutar al inicio del turno o cuando el cliente reporte lentitud. Muestra todos los eventos del día con total de usuarios únicos.
PowerShell ⭐ Usuarios conectados hoy + resumen 
# Ver conexiones RDS de HOY con total de usuarios únicos
$hoy = (Get-Date).Date

Get-WinEvent -LogName "Microsoft-Windows-TerminalServices-LocalSessionManager/Operational" |
Where-Object { $_.Id -in (21,22,23,24,25) -and $_.TimeCreated -ge $hoy } |
Select-Object TimeCreated, 
              @{N="Evento";E={
                switch($_.Id){
                  21 {"✅ Conectado"}
                  22 {"🖥️ Shell iniciado"}
                  23 {"🚪 Logout"}
                  24 {"⚠️ Desconectado"}
                  25 {"🔄 Reconectado"}
                }
              }},
              @{N="Usuario";E={$_.Properties[0].Value}} |
Sort-Object TimeCreated -Descending |
Format-Table -AutoSize

# Total de usuarios únicos conectados hoy (solo evento 21 = login exitoso)
$totalUsuarios = Get-WinEvent -LogName "Microsoft-Windows-TerminalServices-LocalSessionManager/Operational" |
Where-Object { $_.Id -eq 21 -and $_.TimeCreated -ge $hoy } |
ForEach-Object { $_.Properties[0].Value } |
Sort-Object -Unique

Write-Host "════════════════════════════════════════" -ForegroundColor Cyan
Write-Host "📊 RESUMEN DEL DÍA: $(Get-Date -Format 'dd/MM/yyyy')" -ForegroundColor Cyan
Write-Host "════════════════════════════════════════" -ForegroundColor Cyan
Write-Host "👥 Total usuarios únicos conectados hoy: $($totalUsuarios.Count)" -ForegroundColor Green
Write-Host "📋 Usuarios:" -ForegroundColor Yellow
$totalUsuarios | ForEach-Object { Write-Host "   → $_" -ForegroundColor White }
Write-Host "════════════════════════════════════════" -ForegroundColor Cyan

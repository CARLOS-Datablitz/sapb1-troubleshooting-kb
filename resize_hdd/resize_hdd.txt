+ Cuando se edita/agrega memoria RAM desde el PROXMOX, se puede editar con el maquina encendida, pero solo tomará los cambios cuando la reinicie.
**********HDD:
* El HDD si lo puedes editar encendido, no es necesario editar para que tomen los cambios.

Primero ampliar HDD desde PROXMOX, luego lo de abajo:
#!/bin/bash
# Ampliación de disco para XFS (Ubuntu/Linux)
echo "1. Verificando espacio y filesystem..."
lsblk
lsblk -f

echo "2. Instalando cloud-guest-utils (si es necesario)..."
sudo apt update && sudo apt install cloud-guest-utils -y
# verificar si esta instalado con : dpkg -l cloud-guest-utils
echo "3. Ampliando partición..."
# ver si growpart esta instalado: which growpart
sudo growpart /dev/vda 2

echo "4. Expandiendo filesystem XFS..."
sudo xfs_growfs /

echo "5. Verificando..."
df -h /
lsblk

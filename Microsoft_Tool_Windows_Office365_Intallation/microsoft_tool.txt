En el RDS Server:

1. Descargar Office Deployment Tool:
Open in Google browser:
https://www.microsoft.com/en-in/download/details.aspx?id=49117
y luego ejecutar, esto descomprimirá en una carpeta,le das la ubicación en C:/ llamada odt(la creas)

-----------------------------------------


--- Como ver en Windows Server 2019 la arquitectura de CPU ---
cmd  !!!NOT POWESHELL !!!
echo %PROCESSOR_ARCHITECTURE%


2. Create & save your configuration file:

https://config.office.com/deploymentsettings

64 bits
Aplicaciones de Microsoft 365 para negocios(tercero en la lista)
Seleccionar aplicaciones que pide cliente(Excel, Word, etc)
Seleccionar el idioma principal(que usara el cliente, si esta en Alemania será el aleman).
- Licencias y activación
	- Aceptar terminos de licencia
	Basado en usuarios
	General: Colocar el nombre de la compañia <NO PARTNER>
Exportar
	- Mantener configuración actual
	- Aceptar los terminos del contrato de licencia
	- Nombre del archivo: configuration_<customerName>

Mover el archivo generado a al misma carpeta donde se encuentra los otros dos archivos:
C:/odt/
configuration-Office365-x64.xml
setup.exe
configuration_<customer name>.xml
-----------------------------------------
Abrir CMD como Administrator:

cd C:\odt
C:\odt>DIR

setup.exe /configure <customer name>.xml

# al ejecutar lo de arriba se inicia el proceso de instalación

-----------------------------------------
Tomar screeshot de los programs instaldos y enviar a cliente.
